=======================

1. Open Command Prompt as Administrator
   - Press Windows + R
   - Type "cmd"
   - Press Ctrl + Shift + Enter

2. Navigate to the GOOJARA folder
   - Use "cd" command to go to your GOOJARA folder
   - Example: cd C:\Users\YourName\Documents\goojara

3. Install required packages:
   python -m pip install requests beautifulsoup4 lxml selenium

4. If you get permission errors, try:
   python -m pip install --user requests beautifulsoup4 lxml selenium
